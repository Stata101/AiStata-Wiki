Stata基础

🏷️ 基础 · 入门
📖 学习时间: 1-2天

概述

Stata是一款强大的统计分析软件，广泛应用于经济、金融、社会科学等领域。它提供了数据管理、统计分析、图表可视化等功能。

核心特点

  • 命令驱动：使用命令行语法
  • 数据处理：强大的数据管理能力
  • 统计分析：涵盖多种统计方法
  • 可扩展性：支持用户编写程序

Stata界面

主要窗口

  • 命令窗口：输入Stata命令
  • 结果窗口：显示命令执行结果
  • 变量窗口：显示数据集中的变量
  • 数据编辑器：查看和编辑数据
  • 图形窗口：显示绘制的图表
  • 历史窗口：显示之前执行的命令

工作流程

  1. 打开或导入数据
  2. 查看和检查数据
  3. 进行数据处理
  4. 运行统计分析
  5. 保存和导出结果

命令语法

基本语法

// 基本命令结构
[命令] [变量名列表] [, 选项]

// 示例
summarize price weight length

// 带选项的命令
summarize price, detail
histogram price, bin(20) frequency

常用命令前缀

// 命令前缀
bysort group: summarize price  // 分组汇总
levelsof variable, local(levels)  // 提取变量值到局部宏

// 全局和局部宏
global path "C:/data"
local filename "analysis"

// 循环
foreach var in price weight length {
    summarize `var'
}

forvalues i = 1/5 {
    display "Value: `i'"
}

数据处理

数据导入

// Excel文件
import excel "data.xlsx", firstrow clear

// CSV文件
import delimited "data.csv", clear

// Stata格式
use "data.dta", clear

数据导出

// 保存为Stata格式
save "mydata.dta", replace

// 导出为Excel
export excel "results.xlsx", replace

// 导出为CSV
export delimited "results.csv", replace

数据查看

// 查看数据信息
describe
codebook

// 显示数据
list in 1/5
browse

// 查看变量标签
label list

常用命令

数据管理

// 生成新变量
gen log_price = log(price)

// 条件生成
gen high_price = (price > 10000) if !missing(price)

// 重命名变量
rename oldname newname

// 删除变量
drop variable_name

// 保留变量
keep variable1 variable2

统计分析

// 描述性统计
summarize price weight
tabulate foreign
correlate price weight length

// 图形绘制
histogram price
scatter price weight
graph box price, over(foreign)

最佳实践

  1. 使用do文件：编写可重复的分析脚本
  2. 设置工作目录：使用明确的文件路径
  3. 保存日志：记录分析过程
  4. 注释代码：提高可读性
  5. 定期保存：防止数据丢失
  6. 测试小样本：调试代码时使用小数据集

do文件模板

// Stata分析脚本模板

// 设置工作目录
cd "C:/path/to/project"

// 设置日志文件
log using "analysis.log", replace

// 清除内存
clear all
set more off

// 加载数据
use "data.dta", clear

// 数据处理
* 你的数据处理代码

// 统计分析
* 你的分析代码

// 保存结果
esttab using "results.tex", replace

// 关闭日志
log close

// 退出Stata
exit
