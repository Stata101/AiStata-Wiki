数据管理

🏷️ 数据操作 · 基础
📖 学习时间: 1-2天

概述

数据管理是Stata的核心功能之一。有效的数据管理包括数据导入导出、变量操作、数据清洗、数据合并等。

数据管理的重要性

  • 确保数据质量
  • 提高分析效率
  • 保证结果可重复性
  • 便于数据共享

数据导入导出

导入数据

// Excel文件
import excel "data.xlsx", firstrow sheet("Sheet1") clear

// CSV文件
import delimited "data.csv", clear delimiter(",")

// Stata格式
use "data.dta", clear

// 固定格式文本
infix str id 1-5 age 6-7 income 8-15 using "data.txt"

导出数据

// 保存为Stata格式
save "mydata.dta", replace

// 导出为Excel
export excel "results.xlsx", replace firstrow

// 导出为CSV
export delimited "results.csv", replace

变量管理

变量类型

// 字符串变量
str20 name  // 20字符的字符串

// 数值变量
byte age     // 小整数
int income   // 中等整数
long id      // 大整数
float price  // 单精度浮点数
double wage  // 双精度浮点数

变量操作

// 创建变量
gen log_price = log(price)
gen age_sq = age^2

// 条件生成
gen high_income = (income > 50000) if !missing(income)

// 重命名
rename oldvar newvar

// 删除变量
drop var1 var2

// 保留变量
keep var3 var4

// 添加标签
label variable price "商品价格"
label define highlow 0 "Low" 1 "High"
label values high_income highlow

数据操作

egen函数

// 分组统计
bysort group: egen mean_income = mean(income)
bysort group: egen median_income = median(income)
bysort group: egen sd_income = sd(income)

// 其他egen函数
egen total_income = total(income)
egen max_income = max(income)
egen min_income = min(income)
egen count_income = count(income)

数据排序

// 排序
sort age income

// 分组排序
bysort group (age): keep if _n == 1

// 降序排序
gsort -income

合并与追加

追加数据

// 添加观测值
use "data1.dta", clear
append using "data2.dta"
append using "data3.dta"

合并数据

// 一对一合并
use "master.dta", clear
merge 1:1 id using "using.dta"

// 一对多合并
merge m:1 group_id using "group_data.dta"

// 多对一合并
merge 1:m household_id using "member_data.dta"

// 处理匹配结果
tabulate _merge
keep if _merge == 3  // 仅保留匹配成功的
drop _merge

数据重塑

// 宽转长
reshape long income expenditure, i(id) j(year)

// 长转宽
reshape wide income expenditure, i(id) j(year)

最佳实践

  1. 备份原始数据：永远不直接修改原始数据
  2. 使用描述性变量名：避免x1、x2等不明确的名称
  3. 添加变量标签：清楚说明变量含义
  4. 检查缺失值：了解缺失值的模式和原因
  5. 保留数据操作日志：记录所有数据处理步骤
  6. 定期保存中间结果：防止工作丢失
