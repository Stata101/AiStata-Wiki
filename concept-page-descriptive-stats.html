描述性统计

🏷️ 统计分析 · 基础
📖 学习时间: 1天

概述

描述性统计是数据分析的基础，它通过图表和数值指标来总结数据的基本特征。

主要内容

  • 数据的中心趋势
  • 数据的离散程度
  • 数据的分布形态
  • 变量间的关系

描述性统计指标

基本统计量

// 简单统计量
summarize price weight length

// 详细统计量
summarize price, detail

// 分组统计
bysort foreign: summarize price

//  egen 函数计算统计量
bysort foreign: egen mean_price = mean(price)
bysort foreign: egen median_price = median(price)
bysort foreign: egen sd_price = sd(price)

分位数

// 计算分位数
centile price, centile(25 50 75 90)

// 绘制箱线图
graph box price, over(foreign)

// 计算偏度和峰度
estat skewness
estat kurtosis

频数分布

频数表

// 单变量频数分布
tabulate foreign
tabulate rep78
tabulate foreign, missing

// 分组频数表
tabulate foreign rep78

// 数值变量频数分布
histogram price, bin(10) frequency
histogram price, bin(20) density
histogram price, bin(15) percent

累计频率

// 累计频率分布
cumul price, generate(cumul_price)
list price cumul_price in 1/5

交叉表

二维交叉表

// 基本交叉表
tabulate foreign rep78

// 带行百分比
tabulate foreign rep78, row

// 带列百分比和总数
tabulate foreign rep78, column nofreq

// 带卡方检验
tabulate foreign rep78, chi2

// 带Fisher精确检验
tabulate foreign rep78, exact

多维交叉表

// 三维交叉表
table foreign rep78, contents(freq mean price median price)

// 带缺失值的交叉表
tabulate foreign rep78, missing

// 使用egen进行复杂统计
egen avg_price = mean(price), by(foreign rep78)
table foreign rep78, contents(freq avg_price)

图形展示

直方图和密度图

// 直方图
histogram price, bin(15) frequency name(hist1)

// 密度图
kdensity price, generate(x y) nograph
twoway line y x, name(kd1)

// 同时绘制直方图和密度图
histogram price, bin(15) fraction addplot(kdensity) name(combined1)

散点图

// 基本散点图
scatter price weight, name(scatter1)

// 带拟合线的散点图
twoway scatter price weight || lfit price weight, name(fit1)

// 分组散点图
scatter price weight if foreign == 0 || scatter price weight if foreign == 1, ///
    legend(label(1 "Domestic") label(2 "Foreign")) name(grouped1)

其他图表

// 条形图
graph bar (mean) price, over(foreign)

// 饼图
graph pie, over(foreign)

// 时间序列图
tsline price weight

最佳实践

  1. 先探索数据：在进行复杂分析前先了解数据
  2. 图表配合：结合数值指标和图表进行说明
  3. 分组分析：按重要变量分组比较统计量
  4. 检查异常值：识别可能影响结果的极端值
  5. 完整报告：包含中心趋势、离散程度和分布形态

描述性统计报告示例

// 加载数据
sysuse auto, clear

// 创建统计报告
estpost summarize price weight length

// 导出为表格
esttab using "descriptive-stats.tex", replace ///
    cells("mean sd p50 min max count") ///
    stats(N, fmt(%9.0g) labels("Observations"))
